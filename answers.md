### Задание 2

Решение:

Пусть шеф имеет ранг r (r = 1 - самый сильный шеф, r = 80 самый слабый шеф), тогда у шефа есть 80 - r соперников слабее его, всего возможно 79 соперников.

Вероятность выиграть один этап: P = (80 - r)/79.

Т.к. второй этап формируется случайно, то вероятность выиграть оба этапа: P = ((80 - r)/79)^2.

Тогда математическое ожидание числа победителей равно сумме вероятностей для всех участников:
M = sum от r=1 до 80 (((80 - r)/79)^2).

Пусть k = 80 - r, тогда:
M = (1/79^2) * sum от k=0 до 79 (k^2).

M = 168040 / 6241 = 26.9.

Ответ: 26,9

### Задание 3

Решение:

Предположим, что поток машин описывается пуассоновским процессом. 

По условию вероятность появления хотя бы одной машины за 30 минут равна 0.95, значит вероятность того, что ни одна машина не появится за 30 минут равна 1 - 0.95 = 0.05 = e^(-30λ).

Для времени t минут: P = 1 - 0.05^(t/30).

За 10 минут: P = 1 - 0.05^(10/30) = 0.632

За 27 минут: P = 1 - 0.05^(27/30) = 0.933

```python
def is_isomorphic(s: str, t: str) -> bool:
    if len(s) != len(t):
        return False
    map_st = {}
    map_ts = {}
    for cs, ct in zip(s, t):
        if cs in map_st and map_st[cs] != ct:
            return False
        if ct in map_ts and map_ts[ct] != cs:
            return False
        map_st[cs] = ct
        map_ts[ct] = cs
    return True
```

```python
    n = len(nums) + 1  
    expected = n * (n + 1) // 2
    actual = sum(nums)
```

```python
def prime_factors(n: int):
    factors = []
    d = 2
    while d * d <= n:
        while n % d == 0:
            factors.append(d)
            n //= d
        d += 1
    if n > 1:
        factors.append(n)
    return factors
```

Тело функции также представлено в файле solution.py.

Сложность по времени O(sqrt(n)), по памяти O(k) (число простых множителей).
